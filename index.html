115+ Components

Layout, typography, data tables, charts, forms, interactive tabs, accordions, dialogs — all declarative JSON.

⚡

Reactive State

Template expressions like {{ count + 1 }} with 20+ pipes. Auto-update on state changes, no framework needed.

🔌

MCP-Native

display() wraps your component tree as an MCP tool result. First-class CallTool, SendMessage, and UpdateContext actions.

🌐

Zero-Dep Renderer

Vanilla DOM renderer as a single IIFE script tag. Zero dependencies, zero build step for the browser.

🤖

Auto-Renderers

autoTable, autoChart, autoForm, autoMetrics — generate full UIs from raw data in one call.

🐍

Wire-Compatible

Same $prefab v0.2 wire protocol as the Python prefab-ui. TS is a superset — chart formatting extensions are TS-only.
